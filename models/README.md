# Models

This directory holds **serialized trained models** (for example checkpoints, pickles, or other saved artifacts produced by your training or forecasting code).

**Version control:** Model files are **not** committed to this repository. They stay on your machine (or your own storage); only this `README` is tracked so the folder’s purpose stays documented.

**File Formats:** Model files are stored as `.onnx` file types for cross-language compatability.
